# 基于 ROS2 和 Navigation2 的自动巡检机器人

## 项目介绍

本项目基于 ROS2 Humble 和 Navigation2 实现自动巡航巡检机器人功能。

### 功能包说明

- `fishbot_description`：机器人描述文件，包含仿真环境相关配置
- `fishbot_navigation2`：机器人导航功能配置文件
- `fishbot_interfaces`：机器人自动巡检相关自定义接口
- `autopatrol_robot`：自动巡检核心功能实现包

## 使用环境

- 操作系统：Ubuntu 22.04
- ROS 版本：ROS2 Humble
